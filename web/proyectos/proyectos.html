AutosVip

AutosVip es un proyecto hecho para simular un proyecto real de gestión de administración de personal.

Consiste en un CRUD de gestión de personal, donde se pueden agregar diversos registros (automoviles, mecánicos, etc.), modificar o eliminar en una interfaz de usuario intuitiva.

Herramientas utilizadas

  • Java.
  • Java Swing para la interfaz grafica.
  • MySQL como gestor de base de datos utilizando SQL.
  • XAMPP en conjunto a PHPMyAdmin y Apache Server para el despliegue de los componentes e interconexion.

Linux Azure

Este proyecto trata sobre la implementación de un balanceador de carga para interconectar 3 maquinas virtuales en un entorno virtual en la nube (AZURE).

Consiste en la configuración de FTP, MariaDB, FirewallD, Httpd services entre otros, para poder conectarse a la IP de una de estas maquinas y desplegar una página web con base de datos MySQL y mediante PHP.

Herramientas utilizadas

  • Linux (CentOS 7).
  • Conocimiento en redes (HTTP, HTPPS, TCP, UDP, Firewall, etc.).
  • MySQL como gestor de base de datos utilizando SQL.
  • Azure para las maquinas virtuales y balanceador de carga.
  • HTML, CSS, Javascript, PHP para web.

Bambú Gurú (WEB)

Bambú Gurú es un proyecto que consiste en la llevada al mundo real de una de las soluciones de los objetivos de la FUDESO en conjunto con la ONU (Objetivos ONU).

Como idea, cuenta con la planeación y posterior implementación de un sistema de Login donde se podrán ver distintos eventos patrocinados por diversos organismos relacionados y así poder interconectar a gente interesada.

Herramientas utilizadas

  • HTML.
  • CSS.
  • Javascript.
  • Github pages para el host de la web.
  • JQuery.

Bambú Gurú (Android)

Bambú Gurú es un proyecto que consiste en la llevada al mundo real de una de las soluciones de los objetivos de la FUDESO en conjunto con la ONU (Objetivos ONU).

Cuenta con un diseño de base de datos completa y normalizada en cuarta forma normal, un CRUD para usuarios, articulos, productos de un juego relacionado (con el que se busca generar ganancias e invertir en los objetivos).

Herramientas utilizadas

  • Java.
  • Android Studio.
  • MySQL como gestor de base de datos utilizando SQL.
  • PHP para consumir servicios web desde Java.

Feria Virtual

Feria Virtual es un proyecto enfocado a la agilización y comercialización de productos de una feria para evitar pérdidas y así maximizar ganancias además de impulsar tecnológicamente dichos negocios.

Consiste en un CRUD de gestión general, manejo de usuarios con sus roles (vendedor, comprador, repartidor, etc.), manejo de productos (CRUD para estos), manejo de transacciones y cuentas con validaciones de datos.

Herramientas utilizadas

  • HTML.
  • Java Swing para la interfaz grafica.
  • Javascript.
  • MySQL como gestor de base de datos utilizando SQL.
  • JQuery y PHP para manejo web.
  • Vue.jscomo framework de Javascript y CodeIgniter como framework de PHP(implementado en versión final).
